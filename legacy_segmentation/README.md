# legacy_segmentation

This slideshow shows some of the ideas that were attempted here: https://docs.google.com/presentation/d/141NYhUFSCVKBC5l1jSOyYQT7-VEe49M50kmngzEOJTo/edit?usp=sharing

## Contents

- `data/`: legacy inputs, chunk outputs, OCR artifacts, and tree outputs
    - main input data used to create all the other folders is the "audio_sentence_level" and "video_ocr" folders
- `chunking/`: chunk-generation scripts
- `tree_building/`: tree-construction scripts built on chunk outputs
- `retrieval/`: retrieval evaluation and chunk-embedding utilities
